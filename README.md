# Классификатор типов пластика

Веб-приложение для автоматического определения типа пластика по фотографии с использованием нейронной сети ResNet-50.

## О проекте

Данный проект решает реальную экологическую проблему — помогает людям правильно сортировать пластиковые отходы. Ежегодно в России образуется около 3 млн тонн пластиковых отходов, и большая часть из них попадает на свалки из-за неправильной сортировки.

Приложение позволяет загрузить фотографию пластикового изделия и получить информацию о его типе (PET, PE, PC, PP, PS или Others) с рекомендацией по утилизации.

##  Особенности

- **Простой интерфейс** — загрузка фото в один клик
- **Быстрый результат** — ответ за 2-4 секунды
- **6 типов пластика** — распознаёт самые распространённые виды
- **Открытый код** — можно улучшить и адаптировать под свои задачи

##  Технологии

### Backend
- **Python 3.8+** — язык программирования
- **TensorFlow 2.x** — фреймворк для глубокого обучения
- **Keras** — высокоуровневый API для нейронных сетей
- **Flask** — веб-фреймворк
- **NumPy** — работа с числовыми массивами
- **Pillow** — обработка изображений

### Frontend
- **HTML5** — структура страницы
- **Tailwind CSS 3.x** — современный CSS-фреймворк
- **JavaScript (ES6+)** — интерактивность
- **Fetch API** — HTTP-запросы

### Machine Learning
- **ResNet-50** — свёрточная нейронная сеть с 50 слоями
- **Data Augmentation** — аугментация данных для устойчивости модели
- **Transfer Learning** — возможность дообучения с предобученными весами

## 🚀 Установка и запуск

### Требования
- Python 3.8 или выше
- pip (менеджер пакетов Python)
- Ноутбук или компьютер с 8+ ГБ оперативной памяти

### Шаг 1. Клонирование репозитория

```bash
git clone https://github.com/ВАШ_НИК/plastic-classifier.git
cd plastic-classifier
```

### Шаг 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### Шаг 3. Обучение модели

⚠️ **Важно:** Файл обученной модели не загружается на GitHub из-за большого размера (~250 МБ). Необходимо обучить модель самостоятельно.

```bash
python plastic_type_classifier.py
```

Процесс обучения занимает 3-6 часов на обычном ноутбуке (CPU). После завершения будет создан файл `plastic_type_classifier.keras`.

**Альтернатива:** Для ускорения обучения можно использовать предобученные веса ImageNet (требуется VPN или прокси для скачивания).

### Шаг 4. Запуск веб-приложения

```bash
python app.py
```

Откройте в браузере: **http://127.0.0.1:5000**

## Использование

1. Откройте сайт в браузере (на компьютере или смартфоне)
2. Нажмите кнопку **«Выбрать изображение»**
3. Сфотографируйте пластиковое изделие или выберите фото из галереи
4. Нажмите **«Определить тип пластика»**
5. Через 2-4 секунды получите результат с рекомендацией по утилизации

##  Результаты

### Текущие показатели
- **Точность на валидации:** ~25%
- **Скорость инференса:** 2-4 секунды
- **Размер модели:** ~250 МБ

### Почему точность низкая?

Модель обучалась **с нуля** (без предобученных весов ImageNet) на относительно небольшом датасете (~2500 изображений). ResNet-50 содержит около 25 миллионов параметров, и для их полноценной настройки требуются десятки тысяч разнообразных изображений.

### Анализ ошибок

Модель испытывает затруднения при различении:
- **PET и PC** — оба прозрачные, схожая форма
- **PE и PP** — оба непрозрачные, схожая текстура

Это соответствует теоретическим ожиданиям — именно эти пары классов имеют наиболее близкие визуальные признаки.

## Планы по улучшению

3. **Новые категории** — распознавание дополнительных типов пластика
4. **Механизм отказа** — если модель неуверена, предложить пользователю проверить маркировку вручную
5. **Мобильное приложение** — нативное приложение для iOS/Android
6. **Интеграция с картами** — поиск ближайших пунктов приёма вторсырья
