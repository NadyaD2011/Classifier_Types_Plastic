# Как внести вклад в проект

Спасибо за интерес к проекту! Вот несколько правил:

## Стиль кода
- Python: следуйте PEP 8
- Используйте осмысленные имена переменных
- Добавляйте комментарии к сложным местам

## Как предложить изменения
1. Сделайте форк репозитория
2. Создайте ветку для вашей функции (`git checkout -b feature/amazing-feature`)
3. Закоммитьте изменения (`git commit -m 'Add amazing feature'`)
4. Запушьте в ветку (`git push origin feature/amazing-feature`)
5. Откройте Pull Request

## Тестирование
Перед отправкой PR убедитесь, что:
- Код запускается без ошибок
- Модель обучается на тестовых данных
- Веб-интерфейс работает локально
